abstract: This project develops a single-level 2D horizontal shooter game for the Game Boy platform. The game is a wave-based Amogus-themed shooter with progressive difficulty, multiple enemy types, and a boss encounter. The final game is fully playable and demonstrates core Game Boy programming concepts, including sprite management, state machines, collision detection, and audio integration, while delivering a playable arcade experience.
